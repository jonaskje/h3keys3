# H3KEYS3

(Modified version of https://github.com/h3r2tic/h3keys3)

Enables a bunch of programmer friendly short-cuts when holding down CapsLock.

## Usage

- Run h3keys3.exe at startup to enable the shortcuts
- Only tested on a Swedish keyboard layout 
- `Cp` is short for `CapsLock`
- `D` works like a shift modifier and
- `F` works like a control modifier.


## Navigation actions

| Key | Action |
|-----|--------|
|`Cp`+`J`| Down |
|`Cp`+`K`| Up |
|`Cp`+`H`| Left |
|`Cp`+`L`| Right |
|`Cp`+`U`| Home |
|`Cp`+`I`| End |
|`Cp`+`F`+`J`| Page Down |
|`Cp`+`F`+`K`| Page Up |

Pressing `D` with the navigation commands will enable visual selection.
